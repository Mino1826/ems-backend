# EMS Backend

EMS Backend is a Java Spring Boot backend project for practicing employee management APIs and backend development concepts.

This project focuses on REST API design, layered architecture, Maven, and database integration.

## Tech Stack

- Java
- Spring Boot
- Maven
- Spring Data JPA
- MySQL

## Features

- Employee CRUD
- REST API structure
- Maven project setup
- MySQL database connection
- Layered backend architecture
